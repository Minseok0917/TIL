## 회사

어제 기획이 변경되었다. 그렇기에 수정을 해야했다.  
퍼블리싱을 기다리는 시간이 아까워 내가 후딱 개발했다.

개발하는 과정에서 기획에 미흡한 점을 계속 질문해서  
해답을 찾거나 API 관련 이슈 수정요청을 하느라 좀 힘들다.

## 집

디자인을 하느라 시간이 오래걸려 다 완성하진 못했다.
