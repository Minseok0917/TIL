### 1.2 타입 실체 이해 

#### 원시타입 : 하나의 값을 담고 있는것
- undefined
- null
- boolean
- number
- string
- symbol

#### 참조 타입 : 여러개의 값을 담을 수 있는 것
- object

#### null 에 대하여
null 의 타입은 **object** 이다.      
원래는 null 로 나와야 하지만 Javascript Type 버그다.       
수정하지 않는 이유는 변경시 기존 작동하는 소프트웨어가 멈출 수 있기 때문이다.         

``` js
// null 검사 방법
const a = null;
(!a && typeof a === 'object' ); // true 
```
