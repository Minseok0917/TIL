## Vue Router Folder Structure

public 과 private 에서는 meta 데이터에 공개 비공개 구분하는 값을 넣어준다.

-   router
    -   index.js : 권한 검사
    -   routes
        -   index.js : public + private
        -   public.js : 공개
        -   private.js : 비공개
