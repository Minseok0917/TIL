# VitePress Simple Theme

## AutoSidebar 
사이드바 자동추가 되는 기능을 Node 를 이용해서 자동으로 불러오는 기능을 구현했지만     
NAV 의 자식요소를 고려하지 않아 다시 구현해야 될 거 같다.

## Theme Custom
초반에는 테마 커스텀하기 난감했는데      
그냥 `node_modules` 에 있는 vitepress 폴더에 가서 기본테마 코드를 다 가져왔다.      

간단한 UI 레이아웃만 잡고 디자인은 내일 할 생각이다.
