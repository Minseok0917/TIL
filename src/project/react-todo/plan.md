# Plan

## Project Management

### Story
그동안 글을 써 올 때 제목에 날짜를 붙이곤 했다.     
내용을 수정하면 제목의 날짜도 수정해야 됬는데,       
anchor 주소가 바뀌다 보니 이전에 걸어준 주소를 다 바꿔줘야 했다.       

### Solution
제목에 날짜를 붙이지 않고 `release.md` 를 새로 추가하기로 했다.
