# 백준 실버1 달성

하룻동안 11문제를 풀었지만 어려웠던 문제 3개를 푸는데 상당히 오래걸렸다.  
아직 기초강의 1부분에 챕터하나만 본건데 대략 스택에 대한 느낌은 알았다

스택관련 문제를 풀어보다 보니 실버3에서 실버1까지 올라가있었다.  
일요일에는 2챕터의 강의내용을 보고 싶으나 학교에 가야되서 보기 힘들꺼 같다.

그동안 못 읽은 역행자를 가져가서 5챕터를 읽어볼 예정이다
