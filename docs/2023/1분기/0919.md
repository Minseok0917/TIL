## 회사

회사에서는 API 관련 이슈 처리하고 다음 시간에는 디자인 공부했다.  
공부라기보단? 핀터레스트나 awwwards 보면서 여러 종류의 디자인을 봤다.

약간 디자인의 세계란 이런건가? 이런 디자인이 보기가 좋네하면서 구경했다.  
내 목적을 위해서 개발능력도 필요하지만 디자인도 좀 필요하다고 생각한다.
